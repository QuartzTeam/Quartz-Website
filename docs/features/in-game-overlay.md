# In-Game Overlay Fonts

Applies the font you've picked for Quartz's own UI to a few pieces of A Dance of Fire and Ice's native HUD text, so your chosen font shows up during play, not just in the mod's menu. Each piece has its own size control on top of the font.

## Where to find it

These live on the overlay pages in the sidebar, next to the overlay they belong to:

- **Native Title Font** — on the **Overlay → Song Title** page.
- **Countdown Font** — on the **Overlay → General** page.
- **Judgement Popup Font** — on the **Overlay → Judgement** page.

## Highlights

- **Native Title Font** — the level title shown during play.
- **Countdown Font** — the pre-level "3, 2, 1, Go!" countdown.
- **Judgement Popup Font** — the per-hit rating popup ("Perfect!"/"Early!").
- Each one is independent and off by default. Turning it on uses whichever font you've selected for the mod, and a **Font Size** slider scales just that piece.

!!! note
    The **Judgement Popup Font** only re-fonts the per-hit rating popup. It's separate from the [Judgement](judgement.md) overlay, which shows accumulated hit counts.
