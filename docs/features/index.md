# Features

Everything Quartz does lives in one in-game menu, organized into tabs. This section documents each feature area — pages are added and updated as releases ship.

## Overlays & HUD

| Feature | What it does |
|---------|--------------|
| [Key Viewer](key-viewer.md) | On-screen key display with press counters, KPS, rain effects, and DM Note-compatible CSS styling. |
| Combo | Combo counter overlay for gameplay. |
| Judgement | Hit judgement display. |
| Progress Bar | Level progress overlay. |
| Song Title | Shows the current song title in-game. |
| Panels | Draggable stat panels driven by live play statistics. |

## Editor

| Feature | What it does |
|---------|--------------|
| Editor tweaks | Editor readouts (total angle, beats, duration), BGA and difficulty helpers. |

## Visual & Gameplay tweaks

| Feature | What it does |
|---------|--------------|
| Planet Colors | Customize planet colors. |
| Nostalgia | Restore older ADOFAI visuals. |
| Effect Remover | Strip distracting level effects. |
| UI Hider | Hide parts of the game UI. |
| Otto Icon | Customize the Otto icon. |
| Game Overlay Font | Replace the game overlay font. |

## Input & Restrictions

| Feature | What it does |
|---------|--------------|
| Key Limiter | Restrict which keys count as input. |
| Restriction | Additional gameplay restrictions. |
| Chatter Blocker | Block unwanted chat/input interference. |

## Utility

| Feature | What it does |
|---------|--------------|
| Auto Deafen | Discord-integrated automatic deafen while playing. |
| Optimizer | Performance toggles: background execution, process priority, and more. |
| Play Count | Tracks your play counts per level. |

!!! info
    A feature without a linked page hasn't been documented in depth yet. The in-game tooltips cover every setting in the meantime.
