# Otto Icon

Replaces the editor's Otto (auto-play) icon with Quartz's own icon, tinted to whatever color you like.

## Where to find it

Open the Quartz menu in-game and pick **Visuals → Otto Icon** from the sidebar.

## Highlights

- **Otto Color** and **Offset X** / **Offset Y** for basic placement and tint. The icon dims automatically while auto-play is off.
- **Separate High BPM Color** — use a different color while the level's top BPM is 300 or higher, which is where the vanilla icon normally turns red. Off keeps the normal color always.
