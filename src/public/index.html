    Quartz Lite

    Too many settings? Meet Bismuth.

    Bismuth is the lite version of Quartz: the overlays and QoL essentials in one lightweight mod, with a minimalist & polished design philosophy.
    Download Bismuth View on GitHub
    Feature comparison between Quartz and Bismuth
    Feature Quartz Bismuth
    Stats overlays & key viewer ✓Included ✓Included
    Custom fonts ✓Included ✓Included
    Key limiter & chatter blocker ✓Included ✓Included
    Game UI editor & hide-UI toggles ✓Included ✓Included
    Editor tools ✓Included —Not included
    Planet colors, nostalgia & effect remover ✓Included —Not included
    Restriction & auto deafen ✓Included —Not included
    Mod loaders MelonLoader & UMM UMM only

    NoteBismuth supports UnityModManager only. On MelonLoader, install the UMMCompat plugin and put Bismuth under UMMMods/.
